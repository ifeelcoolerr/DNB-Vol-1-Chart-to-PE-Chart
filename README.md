# Vs. Dave and Bambi Volume 1 Chart Converter to Psych Engine
This python script changes volume 1 chart to psych engine lololol

# Instructions
Put a volume 1 chart inside a folder, you should also paste the converter inside it.
You then open the python script, change anything that has "put-song-name-here" with the song name included by the json.
Open command prompt and use "cd" on the directory you are in. paste "python chart_converter.py" and then press enter
You shall now see that a new file is created, that means your chart has succesfully been converted.
You can now use it for Psych Engine!
