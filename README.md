# Vs. Dave and Bambi Volume 1 Chart Converter to Psych Engine
This python script changes volume 1 chart to psych engine lololol

# Instructions
1. Put a volume 1 chart inside a folder, you should also paste the converter inside it.
2. Open command prompt and use "cd" on the directory you are in. paste "python chart_converter.py" and then press enter
3. Enter the song name when prompted.
4. You shall now see that a new file is created, that means your chart has succesfully been converted.
5. You can now use it for Psych Engine!
